GitHub Integration

A Personal Access Token (PAT) is required to upload PDF die lines to the GitHub repository. The token is stored only in your browser.

To create a token: GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token. Select repo scope.
